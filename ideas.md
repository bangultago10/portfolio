# 그림결 포트폴리오 앱 - 디자인 아이디어

## 선택된 디자인 철학

### **Minimalist Elegance with Artistic Depth**

일러스트레이터와 원화가의 작품을 최대한 돋보이게 하면서도, 인터페이스 자체는 세련되고 절제된 미니멀 디자인을 적용합니다. 작품이 주인공이 되고, UI는 그것을 감싸는 우아한 프레임 역할을 합니다.

#### **핵심 원칙**
1. **작품 중심성**: 모든 UI 요소는 사용자의 일러스트와 원화를 최대한 돋보이게 하도록 설계
2. **공간의 미학**: 넉넉한 여백과 정제된 레이아웃으로 고급스러운 갤러리 느낌 연출
3. **타이포그래피 계층**: 명확한 폰트 계층 구조로 정보 우선순위 표현
4. **미묘한 인터랙션**: 부드러운 전환과 호버 효과로 디지털 감각 강화

#### **색상 철학**
- **주요 팔레트**: 순수한 화이트(#FFFFFF), 깊은 블랙(#0A0A0A), 중성 그레이(#F5F5F5, #E5E5E5, #333333)
- **악센트**: 매우 제한적 - 필요시에만 은은한 그레이 톤 변화로 강조
- **감정 의도**: 작품의 색상이 돋보이도록 배경은 최대한 중립적으로 유지. 갤러리의 흰 벽처럼 깔끔함

#### **레이아웃 패러다임**
- **비대칭 그리드**: 한쪽은 네비게이션/정보, 다른 쪽은 대형 이미지 영역
- **세로 흐름**: 스크롤 중심의 자연스러운 정보 전개
- **여백 활용**: 섹션 간 충분한 공간으로 호흡감 있는 구성

#### **시그니처 요소**
1. **얇은 경계선**: 구분 없이 흐르되, 필요시 1px의 미묘한 라인으로 영역 구분
2. **그라데이션 분할선**: 섹션 전환 시 검은색에서 투명으로 페이드되는 구분선
3. **카드 엘리베이션**: 미묘한 그림자로 깊이감 표현

#### **인터랙션 철학**
- **호버 상태**: 배경 밝기 미세 변화, 테두리 강조, 스케일 0.98~1.02 정도의 미묘한 변화
- **클릭 피드백**: 즉각적인 시각적 반응 (배경색 변화, 그림자 강화)
- **전환 애니메이션**: 모든 상태 변화는 0.2~0.3초의 부드러운 트랜지션

#### **애니메이션 가이드라인**
- **페이드 인/아웃**: 요소 등장 시 투명도 0→1 (300ms ease-out)
- **슬라이드**: 이미지 갤러리 전환 시 부드러운 슬라이드 (400ms cubic-bezier)
- **스케일**: 카드 호버 시 1→1.02 (200ms ease-out)
- **색상 전환**: 배경/텍스트 색상 변화는 항상 트랜지션 적용

#### **타이포그래피 시스템**
- **디스플레이 폰트**: Playfair Display (세리프) - 제목, 메뉴 타이틀
- **바디 폰트**: Lato (산세리프) - 본문, 설명, 라벨
- **모노스페이스**: JetBrains Mono - 태그, 메타데이터

**계층 구조**:
- H1: Playfair Display 48px, 700 weight
- H2: Playfair Display 32px, 600 weight
- H3: Playfair Display 24px, 600 weight
- Body: Lato 16px, 400 weight
- Small: Lato 14px, 400 weight
- Label: JetBrains Mono 12px, 500 weight
