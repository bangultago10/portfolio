1️⃣ 프로젝트 실행하기

GitHub Desktop 실행

왼쪽에서 해당 저장소(Repository) 선택

메뉴 → Repository → Open in Terminal

터미널 창에서 아래 명령어 입력:

pnpm dev

표시되는 링크(예: http://localhost:5173)를
 브라우저로 열기

2️⃣ 웹에서 데이터 수정 후 내보내기

사이트에서 원하는 내용을 수정

내보내기 버튼 클릭

다운로드된 파일을 확인

3️⃣ 내보낸 데이터 적용하기

다운로드한 파일 열기

내용 전체 복사 (Ctrl + A → Ctrl + C)

4️⃣ 프로젝트 파일에 붙여넣기

아래 파일 열기:

client/src/lib/defaultData.ts

파일 안의 기존 내용을 전체 선택 후 붙여넣기

⚠️ 주의:

노란 ] 또는 괄호 포함하여 전체를 완전히 교체

일부만 수정하지 말 것

5️⃣ 저장 후 확인

파일 저장:

Ctrl + S

브라우저에서 새로고침 또는
시크릿 창(Incognito)으로 열어서 확인

6️⃣ GitHub에 변경사항 저장하기

GitHub Desktop으로 돌아가기

변경된 파일 목록 확인

하단의 파란 버튼 클릭:

Commit to main

오른쪽 위의 회전 아이콘 버튼 클릭:

Push origin

👉 이 과정이 완료되면 변경사항이 GitHub에 저장됨

💡 추가 팁 (문제 발생 시)

화면이 안 바뀌면 시크릿 창에서 확인

서버가 꺼졌다면 터미널에서 pnpm dev 다시 실행

복사/붙여넣기 시 일부 내용이 빠지지 않았는지 확인
